# Suraksha-Navigator
AI-powered road safety and navigation platform for India featuring smart emergency location sharing, safety-aware route recommendations, construction alerts, hazard detection, and community-powered road intelligence to help travelers make safer routing decisions.
"Aap Ki Suraksha, Aapke Haathon Mein"

Suraksha Navigator is a safety-first navigation platform built specifically for Indian road conditions. The system combines AI-powered route intelligence, government road data, and community-driven reporting to provide travelers with safer and more informed routing decisions.

Key Features:
• Smart Emergency Location Sharing
• AI-Powered Green, Yellow, and Red Route Recommendations
• Road Hazard Detection
• Construction and Road Closure Alerts
• Nearest Police Station Identification
• Community-Based Safety Reporting
• Interactive Map Visualization

Tech Stack:
• React.js
• Node.js & Express.js
• Firebase Firestore
• OpenStreetMap
• Leaflet.js
• Google Gemini / OpenAI APIs

Mission:
To reduce road accidents, improve emergency response times, and provide safety-aware navigation for millions of travelers across India.
